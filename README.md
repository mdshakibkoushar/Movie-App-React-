# 🎬 Movie App (React)

A modern Movie Application built with React.js that fetches movie data from an API and displays trending movies with details like ratings, release date, overview, and trailers.

## 🚀 Features

- 🔍 Search Movies
- 🎬 Trending Movies List
- 📅 Release Date
- ⭐ Ratings
- ❤️ Add to Favorites
- ▶️ Watch Trailers
- 📱 Fully Responsive UI

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- CSS / Tailwind (if used)
- Movie API

## 📦 Installation

```bash
git clone https://github.com/mdshakibkoushar/Movie-App-React-.git
cd Movie-App-React-
npm install
npm run dev
